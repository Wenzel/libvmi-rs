## Overview

### Design goals

- **Safety**: _LibVMI_ runs with very high privilege and in dangerous contexts
  such as malware analysis
- **Portability**: _LibVMI_ aims to provide a standard VMI API accross hypervisors
  on Windows/Linux/MacOS
